bool is_time_to_purge ( void  )

Called in main() to determine whether or not purge() was run today, reads and writes to the 'lastpurge` file. If it hasn't been run today, the current day will be written. If 'lastpurge' doesn't exist, it gets created.

Returns
FALSE if the day is the same (don't purge); otherwise TRUE and writes the current day to 'lastpurge'

Detailed Description

Each waste directory is added to a linked list and has the data from this structure associated with it.

Field Documentation

◆ dev_num

unsigned int st_waste::dev_num

The device number of the filesystem on which the file resides. rmw does not copy files from one filesystem to another, but rather only moves them. They must reside on the same filesystem as a WASTE folder specified in the configuration file.
